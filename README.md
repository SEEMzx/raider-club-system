# Raider Club System

一个基于 **Flask** 的游戏战队管理系统，用于高效管理战队成员、记录对战数据并生成排行榜。

---

## ✨ 功能特点

- **用户管理**：支持管理员、队长、成员等角色权限管理。
- **战队管理**：可创建、加入或退出战队，管理队员信息。
- **战绩系统**：支持对战记录提交、审核与战绩截图上传。
- **实时排行榜**：包括个人 KD 和团队统计数据。
- **数据可视化**：提供战绩统计的图形化展示。
